core:autocomplete
-----------------

* Description: Create file for Bash autocomplete
* Usage:

  * `core:autocomplete`

The <info>core:autocomplete</info> creates a script to autocomplete Gush commands in Bash:

    <info>$ gush core:autocomplete</info>
